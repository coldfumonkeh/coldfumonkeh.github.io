allVideosByArtist
public any allVideosByArtist ( required string artist_uri, string returnQuery="no" )

Output: enabled
Parameters:
   artist_uri: string, required, artist_uri - the unique uri link for the artist (eg: http://api.mtvnservices.com/1/artist/[artist_uri]/videos/)
   returnQuery: string, optional, returnQuery - if yes, the results will be returned as a query recordset. Default is 'no'.

artistBrowse
public any artistBrowse ( required string browseAlpha, string returnQuery="no" )

Output: enabled
Parameters:
   browseAlpha: string, required, browseAlpha - a single character, required for searching. ('a-z', or '-' for other characters)
   returnQuery: string, optional, returnQuery - if yes, will convert the output to a query format. Default is 'no'.

public any getVideoByID ( required string videoID, string returnQuery="no" )

Output: enabled

   returnQuery: string, optional, returnQuery - if yes, the results will be returned as a query recordset. Default is 'no'

searchArtist
public any searchArtist ( required string searchTerm, numeric maxResults="1", string includeRelated="no", string returnQuery="no" )

Output: enabled
Parameters:
   searchTerm: string, required, searchTerm - the name of the artist/band to search for. The function will escape any characters and URLEncode the term for you
   maxResults: numeric, optional, maxResults - the maximum number of results you want returned
   includeRelated: string, optional, includeRelated - if yes, an extra struct is included, which holds the details of artists/bands related to the selected artist
   returnQuery: string, optional, returnQuery - if yes, the results will be returned as a query recordset. Default is 'no'.
